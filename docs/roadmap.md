# Roadmap

## Phase 1 — Prototype

- Static map and searchable directory
- JSON data file
- Manual photo references
- Basic responsive UI

## Phase 2 — Real Data

- Seed with Southern Maryland / DMV open-play locations
- Add sources and last-verified dates
- Create repeatable research workflow for adding new cities

## Phase 3 — Submission Workflow

- Add “submit a court/update” form
- Store submissions in a database or Airtable/Google Sheet
- Add moderation queue

## Phase 4 — Public Product

- SEO-friendly pages for each city and park
- “Open play near me” geolocation
- User photos and verified updates
- Sponsorship / Scoop Pickleball tie-ins
